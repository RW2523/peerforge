## Event Envelope Contract

### WebSocket Message Types

**Event Envelope** (server → client):
```typescript
{
  type: 'agent_message' | 'intervention' | 'presence_update' | 'typing' | 'state_update',
  debate_id: string,
  sequence_number: number,
  event_id: string,
  occurred_at: string,  // ISO 8601
  sender_type: 'system' | 'user' | 'agent',
  sender_id: string | null,
  payload: { ... }  // event-specific data
}
```

**Command Message** (client → server):
```typescript
{
  command: 'join_presence' | 'leave_presence' | 'typing' | 'control.pause' | 'control.resume' | 'control.next_turn' | 'control.end' | 'intervene',
  debate_id: string,
  request_id: string,  // client-generated for ACK tracking
  payload?: { ... }  // command-specific data
}
```

**ACK Message** (server → client):
```typescript
{
  type: 'ack',
  request_id: string,
  command: string,
  timestamp: string
}
```

**ERROR Message** (server → client):
```typescript
{
  type: 'error',
  request_id: string,
  command: string,
  error: string,  // human-readable error message
  timestamp: string
}
```

## Known Limitations

### Production Deployment Notes

1. **WebSocket URL Configuration**
   - Frontend defaults to `ws://localhost:8000` for development
   - Set `NEXT_PUBLIC_WS_URL` environment variable for production (e.g., `wss://api.example.com`)
   - Ensure reverse proxy/load balancer supports WebSocket upgrade

2. **SSE Endpoint Status**
   - SSE endpoint (`/debates/{debate_id}/stream`) still exists but deprecated
   - Room UI no longer uses it
   - Safe to remove in future milestone after monitoring confirms no legacy clients

3. **Test Suite Performance**
   - Full pytest suite slow due to DB connection overhead (60+ seconds)
   - Individual test modules pass quickly (<2s)
   - Recommend CI pipeline optimization or parallel test execution

4. **Browser Compatibility**
   - WebSocket client uses native `WebSocket` API (IE 11+ support)
   - Auto-reconnect tested in Chrome/Firefox/Safari
   - Heartbeat prevents idle connection drops on restrictive networks
