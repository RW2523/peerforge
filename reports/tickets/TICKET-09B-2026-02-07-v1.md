## User Can Now Do (Plain English)

1. **Settings**: 
   - Click "Settings" in navigation
   - Enter OpenRouter API key
   - Choose where to save it (memory/session/device)
   - See available credits immediately

2. **Create Debate**:
   - Click "Debates" in navigation
   - Fill in title and problem statement
   - Click "Create & Open Room"
   - Automatically redirected to Room

3. **Continue Debate**:
   - Click "Debates" in navigation
   - See list of all debates (newest first)
   - Each debate shows: title, status (Draft/Live/Complete), last updated time
   - Click "Continue" to open in Room
   - OR click "Operator Console" for backend controls

4. **Seamless Flow**:
   - Settings → Enter key → See credits → Done
   - Debates → Create new → Room → Start debate
   - Debates → Continue old → Room → Resume
   - No confusing detours or duplicate pages
